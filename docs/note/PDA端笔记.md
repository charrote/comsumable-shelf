PDA端聚焦几个业务：
0.首页（与现在一样）
1.收料入库（流程：扫描料盘条码->自动识别大厂商条码格式提取客户料号、数量、规格、批次等信息，通过客户料号、规格自动匹配内部料库料号，匹配不到就是新料号->画面给用户确认各个提取信息->确认保存->打印内部标签）
2.料盘上架
2.1 智能料架上架（流程：扫描Reel标签 -> 放入料架储位 -> 检测到料架储位占用 -> PDA显示Reel/料架/储位 信息 -> 保存）
2.2 手动上架（流程：扫描Reel标签 -> 扫描或选择料架 -> 扫描或选择储位 -> 保存完成） 
参考:docs/manual/02.上架操作说明.md
3.扫码出库
3.1 生产（按Bom单出库）：新建出库料单（选择BOM/填入数量）-> 生成发料单 -> FIFO计算 -> LED亮灯分配 -> 扫码取料->完成
3.2 单独出料：扫Reel -> 保存后出库
4.库存跟踪(与现在一样）
5.设置（与现在一样）
