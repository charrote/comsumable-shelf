二、MES 接口文档
表格
项目	内容
接口地址	示例：http://127.0.0.1:8900/publicApi/upload
接口类型	WebApi
提交方式	post
参数格式	json
1. 请求实例
json
{
  "reelID": "SMT-REPAIR01",   //唯一码
  "Qty": 5000                  //点数数量
}
2. 应答实例
json
{
  "success": true,      //成功或者失败标识
  "code": 0,
  "message": "OK:上传成功"
}
三、业务流程
流程走向
开始 → 放料盘 → 读码 → 点数 → 更新条码中的数量
结果为失败：进入 NG 处理
结果为成功：提交料架 → 打印新标签
标签规则示例
原始条码：4500067189 NI000035181600 030000010
点数数量：2999
新打印标签内容：4500067189 NI000035 181600029990010
