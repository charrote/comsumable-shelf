# 七鑫智能物料架系统 · 需求说明书

## 1. 项目概述
- **使用角色**：仓库操作人员（追求简单、快速、抗疲劳）
- **操作终端**：安卓 PDA 或安卓手机（核心交互为扫码）
- **硬件环境**：智能料架（亮灯/找货）、XR 电子点料机
- **核心目标**：实现物料从入库→智能发料（亮灯/FIFO）→退库点计→账务日清的全流程数字化与防呆管理。

## 2. 核心业务需求

### 2.1 原材料入库
- **扫码录入**：扫码入库，以实际物料编码入库。
- **唯一标识**：以“上架时间”作为该物料盘的唯一定位标识（即使物料编码重复，同样一盘物料扫了两次需防呆）。

### 2.2 BOM 发料管理
- **BOM 导入**：支持导入客户 BOM 表。
- **替代料处理**：系统需解析 BOM 表并处理替代料关系。
- **亮灯指令**：发料指令下发至智能料架，触发对应货架亮灯。

### 2.3 智能料架与先进先出 (FIFO) 出库
- **硬件联动**：系统按出库单自动驱动对应货架亮灯指引。
- **出库策略**：按需求数量出库，物料存在数量大小不等的情况。
    - **策略 A（尾数优先）**：优先出尾数盘。逻辑：尾数不够 → 选大盘 → 大盘不够 → 大盘+小盘。目标：减少尾数盘滞留，降低退库盘量。
    - **策略 B（时间 FIFO）**：不按盘量拆分，严格按“二次入库时间戳”排序，先退库的先出，累计数量达标即止。
    *(注：具体主用策略需客户确认)*

### 2.4 库存账务管理
- **实时更新**：出入库数据实时刷新。
- **客户分类**：支持按“客户别”分类统计。
- **日报表**：账目按日期显示 `在库数量` / `在线数量` / `结余数量`，日日更新。

### 2.5 物料退回与二次入库
- **点料机对接**：对接 XR 点料机，点数上传系统。
- **唯一性匹配**：因物料盘编码无唯一性，系统需自动抓取该物料最后一次出库时间，并以前后 1~5 秒为基准生成二次入库时间戳，完成配对上架。
- **二次上架**：点数完成后，继续扫码上架。

### 2.6 异常出仓与防呆
- **非标准出仓**：支持不够物料时的申补流程。
- **上架防呆**：拦截重复扫码（防重码）。
- **出库防呆**：同一物料盘禁止重复扫码出库。
