# 七鑫智能物料架系统 · PC Web 端 UI/UX 设计

> 读者对象：前端开发、UI/UX 设计师、PM

## 3. 页面列表

| 页面 | 路由 | 权限 | 说明 |
|------|------|------|------|
| 工作台 | `/` | 所有人 | 快捷入口 + 当日概况 |
| 入库列表 | `/inbound/list` | 操作员+ | 入库单管理 |
| 出库管理 | `/outbound` | 操作员+ | BOM → 发料 → 出库 |

| 用户管理 | `/system/users` | 管理员 | 角色分配 |

## 6. 权限模型

| 角色 | 权限范围 |
|------|---------|
| **管理员** | 全部页面 + 系统配置 + 用户管理 |
| **主管** | 全部业务页面 + 查看报表 + 用户管理 (仅查看) |
| **操作员** | 入库/出库/退库/库存查询/报表 (仅查看) |
| **只读** | 库存查询/报表 (仅查看, 无导出) |

**实现方式：** React Router `useRoutes` + 路由守卫 + Ant Design `Button` 权限控制
