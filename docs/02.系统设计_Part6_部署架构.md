```dockerfile

# 创建数据目录
```

## 3. 数据库设计

### 3.1 存储策略

| 阶段 | 存储 | 说明 |
|------|------|------|
| v1 单机 | SQLite + WAL 模式 | 高性能, 单文件, 工厂内网首选 |
| v2 多厂 | PostgreSQL 15+ | 多租户, 远程访问 |
| 归档 | SQLite dump → CSV | 历史数据导出 |

### 3.2 SQLite 优化

```sql
-- WAL 模式 (写入并发)
PRAGMA journal_mode = WAL;
PRAGMA synchronous = NORMAL;

-- 页面大小优化
PRAGMA page_size = 4096;

-- 缓存
PRAGMA cache_size = -64000;  -- 64MB

-- 自动分析
PRAGMA auto_vacuum = INCREMENTAL;
```

### 3.3 数据备份

```bash
#!/bin/bash
# backup.sh - 每日自动备份
BACKUP_DIR="/app/backups"
DATE=$(date +%Y%m%d)
DB="/app/data/smes.db"

# 冷备份 (SQLite 支持热备)
cp "$DB" "$BACKUP_DIR/smes_${DATE}.db"
gzip "$BACKUP_DIR/smes_${DATE}.db"

# 保留 30 天
find "$BACKUP_DIR" -name "smes_*.db.gz" -mtime +30 -delete
```

## 4. 网络与通信

### 4.1 端口规划

| 服务 | 端口 | 协议 | 说明 |
|------|------|------|------|
| FastAPI | 8080 | HTTP | PDA 连接 |
| Modbus TCP | 502 | TCP | 料架主控板 |
| WebSocket | 8081 | WS | 实时亮灯推送 |

### 4.2 心跳与重连

```python
class HeartbeatManager:
    """心跳管理"""
    
    MODBUS_TIMEOUT = 2.0          # 502 端口超时
    SLOT_POLL_INTERVAL = 1.0      # 储位轮询间隔
    LED_RETRY_COUNT = 3           # LED 重试
    LED_RETRY_DELAY = 1.0         # LED 重试间隔
    
    def check_modbus_health(self) -> bool:
        """检查 Modbus TCP 连接"""
        try:
            client = AsyncModbusTcpClient(ip, port=502)
            result = client.read_holding_registers(60000, 1, slave=200)
            return not result.isError()
        except:
            return False
```

## 5. 监控与告警

### 5.1 健康检查端点

```python
@app.get("/health")
async def health_check():
    return {
        "status": "healthy",
        "database": "connected",
        "modbus": "connected" if modbus_healthy else "disconnected",
        "uptime_seconds": time.time() - startup_time,
        "slot_polling": "active" if slot_reader_running else "stopped",
        "led_worker": f"active ({led_queue_size} pending)"
    }
```

### 5.2 监控指标

| 指标 | 类型 | 阈值告警 |
|------|------|----------|
| Modbus 连接状态 | Gauge | disconnected > 30s |
| 储位轮询延迟 | Histogram | > 2s |
| LED 指令队列长度 | Gauge | > 100 |
| LED 指令失败率 | Counter | > 5% |
| 数据库写入延迟 | Histogram | > 100ms |
| PDA API 响应时间 | Histogram | > 1s |
| 缓存未命中率 | Gauge | > 20% |

### 5.3 日志结构

```
logs/
├── app.log           # 应用日志 (结构化 JSON)
├── modbus.log        # Modbus 通信日志
├── led.log           # LED 指令日志
├── slot.log          # 储位轮询日志
└── error.log         # 错误日志 (单独文件)
```

## 6. 安全

| 层级 | 措施 |
|------|------|
| 网络 | 工厂内网隔离, 防火墙限制 |
| API | JWT 认证, 1小时过期 |
| 数据 | SQLite 文件权限 600 |
| 传输 | 局域网 HTTP (v1) / HTTPS (v2 可选) |
| 操作日志 | 所有操作记录 operator + 时间戳 |
| 主管授权 | 防呆拦截需主管扫码/密码 |
