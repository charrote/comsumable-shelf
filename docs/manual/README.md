## 使用说明

本文档目录按业务操作流程组织，每个文件描述一个完整的业务场景。

阅读建议：
- **新操作员**：按编号顺序阅读 01 → 02 → ... → 06
- **熟练操作员**：按需查阅特定操作
- **工程师/管理员**：关注每个文档末尾的「数据流参考」和「异常处理」章节

## 操作流程全景图

```
                          ┌──────────────┐
                          │  客户来料     │
                          └──────┬───────┘
                                 ▼
                     ┌─────────────────────┐
                     │  01. 收料           │  ← OCR识别 / 人工确认 / 自动创建
                     │  (扫码→匹配→入库)   │  ← 打印内部标签
                     └──────────┬──────────┘
                                ▼
                     ┌─────────────────────┐
                     │  02. 上架           │  ← 感应自动 / 手动分配
                     │  (Reel→储位绑定)    │
                     └──────────┬──────────┘
                                │
              ┌─────────────────┼─────────────────┐
              ▼                 ▼                  ▼
   ┌──────────────────┐ ┌──────────────┐ ┌──────────────────┐
   │  03. BOM上传     │ │  05. 单独出库 │ │  06. 点料       │
   │  (Excel→BOM结构) │ │  (扫盘即出)   │ │  (XR→配对→上架)  │
   └────────┬─────────┘ └──────────────┘ └──────────────────┘
            ▼
   ┌──────────────────┐
   │  04. 料单出库    │
   │  (FIFO+LED→取料) │
   └──────────────────┘
```
