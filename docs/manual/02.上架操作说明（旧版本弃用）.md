# 上架操作说明

> 适用端：PDA
> 对应角色：仓库操作员
> 最后更新：2026-06-17
> 关联版本：v1.0

---

## 一、业务目标

将已收料的库存盘（Reel）放入智能料架的指定储位，并在系统中记录"哪个盘在哪个架子的哪个储位上"。

上架完成后：
- 库存盘关联到具体的 **料架编号 + 储位编号**
- 可通过库存查询精确定位到物理位置
- 出库时可控制对应储位 **亮灯引导** 取料

---

## 二、上架方式

根据硬件配置不同，支持两种上架方式：

| 方式 | 硬件要求 | 操作复杂度 |
|------|---------|-----------|
| **感应自动上架**（推荐） | 料架配有 AMKN8702G 主控板 + 储位传感器 | 最低 — 放入即完成 |
| **手动分配储位** | 无传感器料架 | 需操作员扫码 + 指定储位 |

---

## 三、操作流程

### 3.1 前置条件

| 条件 | 说明 |
|------|------|
| 收料已完成 | 库存盘已创建（`status = on_shelf`），但 `shelf_slot_id` 可能为空 |
| 空储位可用 | 目标料架有未占用的储位 |
| 料架通信正常 | Modbus TCP 连接可达（IP + Port 502） |

### 3.2 感应自动上架（推荐）

此流程适用于配有储位传感器的智能料架。

#### Step 1：扫描 Reel

| PDA 操作 |
|----------|
| 上架 → 扫描 Reel 上的标签条码（刚打印的内部标签） |
| 系统查询该 Reel 信息并进入"待上架"状态 |

**PDA 显示**：
```
┌───────────────────────────┐
│  待上架                     │
│  Reel: #1024               │
│  物料: RES-0402-1001       │
│  数量: 1 盘                 │
│  状态: 等待放入料架...      │
└───────────────────────────┘
```

#### Step 2：放入料架储位

操作员将 Reel 放入任意空储位。

料架传感器检测到储位状态从"空"变为"占用"，自动将变化数据上报系统。

> 系统通过 Modbus TCP 轮询检测传感器状态变化，
> 或由主控板主动上报（取决于固件版本）。

#### Step 3：系统自动绑定

| 系统行为 |
|----------|
| 1. 检测到 Slot N 从空变占用 |
| 2. 记录 `ShelfSlotEvent`（event_type=occupied） |
| 3. 自动将当前待上架的 Reel 绑定到该储位 |
| 4. 更新 `InventoryReel.shelf_slot_id = Slot N.id` |

**PDA 显示**：
```
┌───────────────────────────┐
│  ✅ 上架成功               │
│  Reel: #1024               │
│  料架: SH-A-01             │
│  储位: A-面-第5位           │
└───────────────────────────┘
```

---

### 3.3 手动分配储位

适用于**无传感器料架**或**自动上架失败**时的备选方案。

#### Step 1：扫描 Reel

操作同 3.2 Step 1。

#### Step 2：选择目标储位

| PDA 操作 |
|----------|
| 系统显示可用空储位列表（或手动输入储位编号） |
| 操作员选择或扫码储位编号 → 确认 |

#### Step 3：系统绑定

| 系统行为 |
|----------|
| 1. 校验储位是否为空 |
| 2. 校验储位容量是否足够 |
| 3. 绑定 Reel 到储位 |
| 4. 更新 `InventoryReel.shelf_slot_id` |

#### 也可在 PC Web 上操作：

| PC Web 操作 |
|------------|
| 入库收料 → 入库单详情 → 找到目标明细行 → 点击"分配储位" |
| 选择料架 → 选择面(A/B) → 选择储位号 → 确认 |

## 四、数据流参考

### 4.1 涉及的表

| 表 | 感应自动 | 手动 |
|----|---------|------|
| `inventory_pallets.shelf_slot_id` | 自动更新 | 手动更新 |
| `shelf_slot_events` | INSERT (event_type=occupied) | 可选记录 |
| `shelves` / `shelf_slots` | 只读查询 | 只读查询 |

### 4.2 上架后的库存状态

```
InventoryReel
  ├── status = "on_shelf"         ← 上架前后不变（创建时就是 on_shelf）
  ├── shelf_slot_id = 42          ← 绑定到具体储位
  ├── material_id → MaterialMaster
  └── pallet_barcode = "ABC-123"  ← 原始条码

ShelfSlot (id=42)
  ├── shelf_id → Shelf (code="SH-A-01")
  ├── side = "A"
  ├── board_address = 1
  └── slot_on_board = 5
```

## 五、异常处理

| 异常 | 系统行为 | 操作员处理 |
|------|---------|-----------|
| 储位已被占用 | 拒绝绑定，提示"该储位已被占用" | 选择其他空储位 |
| 储位容量不足 | 拒绝绑定，提示"超过储位容量 X 盘" | 选择容量更大的储位 |
| 传感器无响应 | 自动上架超时，回退为手动分配 | 使用手动分配流程 |
| Reel 已绑定储位 | 提示"该盘已上架" | 如需移动，先解绑再重新上架 |
| 放入后传感器未检测到 | 系统可能轮询延迟（最多 2 秒） | 稍等刷新，或手动触发检测 |

## 六、常见问题

**Q: 感应自动上架时，怎么知道系统已经绑定了？**
A: PDA 上会显示"上架成功"提示。如果没有提示，说明传感器可能未检测到，请稍候或检查 Reel 是否完全放入储位。

**Q: 一个储位可以放多盘吗？**
A: 默认一个储位绑定一盘。如果启用了多盘模式（需系统配置），同一储位可放多盘，但在出库亮灯时会同时亮起。

**Q: 上架后想换到另一个储位怎么办？**
A: 在库存管理中找到该盘 → 修改储位 → 选择新的空储位。原储位自动释放。

**Q: 料架上的 LED 灯不亮怎么办？**
A: 检查料架电源和 Modbus 通信。可在系统设置中测试指定储位的 LED 亮灭。
