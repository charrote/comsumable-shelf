# BOM 上传操作说明

> 适用端：PC Web
> 对应角色：工程师 / 计划员
> 最后更新：2026-06-17
> 关联版本：v1.0

---

## 一、业务目标

将客户的 BOM（物料清单）Excel 文件导入系统，完成两件事：
1. **维护 BOM 结构** — 供后续发料单（Issue Order）生成使用
2. **更新物料主数据** — BOM 中出现的物料编码若系统中不存在，自动创建

---

## 二、BOM 文件格式要求

### 2.1 Excel 格式

| 列 | 列名（示例） | 必填 | 说明 |
|----|------------|------|------|
| A | 成品编码 | 推荐 | 如 `PROD-A-001`，BOM 的归属产品 |
| B | 物料编码 | **必填** | 子件物料编码，如 `RES-0402-1001` |
| C | 数量 | **必填** | 单个成品用量，如 `100` |
| D | 单位 | 可选 | 默认 `盘` |
| E | 替代料编码 | 可选 | 可替代的物料编码（1:1 替代） |

### 2.2 示例

| 成品编码 | 物料编码 | 数量 | 单位 | 替代料编码 |
|---------|---------|------|------|-----------|
| PROD-A-001 | RES-0402-1001 | 100 | 盘 | RES-0402-1002 |
| PROD-A-001 | CAP-0603-104 | 50 | 盘 | |
| PROD-A-001 | IC-LM358 | 20 | 盘 | IC-LM358A |

> 第一行为表头，从第二行开始解析。

## 三、操作流程

### Step 1：准备 BOM 文件

按上述格式准备 Excel 文件（`.xlsx` 或 `.xls`）。

### Step 2：上传 BOM

| PC Web 操作 |
|------------|
| BOM 管理 → 点击"上传 BOM" |
| 选择客户（若系统配置了多客户） |
| 选择文件 → 点击上传 |

### Step 3：查看解析结果

上传完成后，系统显示：

```
┌──────────────────────────────────────┐
│  ✅ BOM 上传成功                      │
│                                       │
│  BOM 名称: PROD-A-001_BOM.xlsx        │
│  总行数: 156 行                       │
│  唯一物料: 87 种                      │
│  替代料关系: 12 对                     │
│  自动创建物料: 3 种（新物料）           │
│                                       │
│  [查看BOM明细]  [生成发料单]           │
└──────────────────────────────────────┘
```

### Step 4：验证 BOM 内容

点击"查看BOM明细"确认数据正确：

| 物料编码 | 物料名称 | 数量 | 单位 | 替代料 |
|---------|---------|------|------|--------|
| RES-0402-1001 | 贴片电阻 100Ω | 100 | 盘 | RES-0402-1002 |
| CAP-0603-104 | 贴片电容 0.1μF | 50 | 盘 | — |
| IC-LM358 | 运算放大器 | 20 | 盘 | IC-LM358A |

> 物料名称从 `MaterialMaster` 中读取。若为自动创建的新料，名称默认等于编码，可在物料管理中修改。

## 四、自动创建物料说明

### 4.1 触发条件

BOM 上传时，系统会逐一检查每个物料编码：

```
遍历 BOM 中所有物料编码 →
  检查 MaterialMaster 中是否存在相同 code →
    存在 → 使用已有记录
    不存在 → 自动创建新记录（unit="盘", active=1）
```

### 4.2 创建规则

| 字段 | 值 | 说明 |
|------|----|------|
| `code` | BOM 中的物料编码 | 直接使用 |
| `name` | 同 `code` | 可在物料管理中修改 |
| `unit` | `盘` | SMT 默认单位 |
| `customer_id` | 上传时选择的客户 | 用于隔离客户数据 |
| `active` | 1 | 启用 |

### 4.3 关闭自动创建

如需关闭此功能，在系统设置中将 `BOM_AUTO_CREATE_MATERIAL` 设为 `false`。
此时 BOM 中出现新物料编码会报错提示，不会自动创建。

---

## 五、生成发料单

BOM 上传成功后，可立即生成发料单：

1. 在 BOM 明细页面点击"生成发料单"
2. 选择客户（通常与 BOM 所属客户一致）
3. 输入需求日期（可选）
4. 确认 → 系统创建发料单 `IS-YYYYMMDD-NNN`

> 发料单生成后状态为 `pending`，还需经过 **FIFO 计算 + LED 分配** 才能开始拣料出库（参见《料单出库操作说明》）。

## 六、数据流参考

### 6.1 涉及的表

| 表 | 操作 | 说明 |
|----|------|------|
| `bom_headers` | INSERT | BOM 主表 |
| `bom_details` | INSERT (批量) | BOM 明细行 |
| `material_master` | INSERT (条件) | 仅自动创建新物料时 |
| `issue_order` | INSERT | 仅当生成发料单时 |
| `issue_detail` | INSERT | 发料单明细行 |

### 6.2 解析规则

- 跳过空行
- 跳过数量 ≤ 0 的行
- 物料编码去重后统一检查是否需要创建
- 替代料编码不自动创建（需提前在物料主数据中存在）

## 七、常见问题

**Q: 上传后提示"文件格式不支持"？**
A: 仅支持 `.xlsx` 和 `.xls` 格式。CSV 文件需先在 Excel 中另存为 `.xlsx`。

**Q: BOM 中的物料编码和系统里的编码不同怎么办？**
A: 两种方式：
   - 在 BOM Excel 中直接使用系统中的编码（推荐）
   - 上传后系统自动用 Excel 中的编码创建新物料

**Q: 上传后发现 BOM 有错怎么办？**
A: 可删除该 BOM（需确认没有关联的发料单），修正文件后重新上传。

**Q: 如何修改 BOM 中的物料数量？**
A: 当前不支持直接编辑。需删除后重新上传，或通过新建版本的方式。
