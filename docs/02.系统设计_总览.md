## 技术栈总表

| 层级 | 技术 | 版本 | 说明 |
|------|------|------|------|
| **前端** | React Native (Expo) | SDK 50+ | Android PDA, 全屏 Kiosk |
| **后端** | Python 3.11 + FastAPI | 0.104+ | 轻量、异步、AI 扩展 |
| **数据库** | SQLite (WAL) | 3.40+ | 单机部署首选 |
| **硬件通信** | pymodbus (TCP) | 3.6+ | Modbus TCP 主控板 |
| **消息推送** | WebSocket | 12+ | 实时亮灯状态同步 |
| **BOM 解析** | openpyxl | 3.1+ | Excel 解析 |
| **认证** | python-jose + passlib | 3.3+ / 1.7+ | JWT Bearer |
| **容器** | Docker Compose | 3.8 | 工厂内网离线部署 |
| **日志** | structlog | 23.2+ | 结构化 JSON 日志 |

## 数据库表清单 (16 张表)

| 表名 | 用途 | 关键字段 |
|------|------|----------|
| `customers` | 客户 | code (唯一) |
| `material_categories` | 物料分类 | customer_id + code (唯一) |
| `material_master` | 物料主数据 | customer_id + code (唯一) |
| `material_alternative` | 替代料关系 | customer_id + 原编码 + 替代编码 |
| `shelves` | 智能料架 | code (唯一) |
| `shelf_slots` | 储位 | side + slot_on_board (唯一) |
| `inventory_pallets` | 物料盘库存 | status (on_shelf/tracking/exhausted) |
| `receipt` | 入库单 | receipt_no (唯一) |
| `receipt_pallets` | 入库盘记录 | barcode |
| `issue_order` | 发料出库单 | order_no (唯一) |
| `issue_detail` | 发料明细 | pallet_ids (JSON) |
| `transactions` | 交易流水 (不可变) | type (in/out/restock/reverse) |
| `led_commands` | 亮灯指令队列 | status (queued/sent/cleared/failed) |
| `xr_batches` | XR 点料机批次 | status (pending_match/matched/failed) |
| `bom_headers` | BOM 主表 | parsed 标志 |
| `bom_details` | BOM 明细 | alternate_code |
| `system_settings` | 系统配置 | key-value (JSON) |

## API 路由总览

| 前缀 | 方法 | 端点 | 说明 |
|------|------|------|------|
| `/auth` | POST | `/login` | 用户登录 |
| `/auth` | GET | `/me` | 当前用户 |
| `/materials` | GET | `/?customer_id=&search=` | 物料列表 |
| `/materials` | POST | `/import` | 批量导入物料 |
| `/receipts` | POST | `/*` | 入库 (创建/扫码/确认) |
| `/bom` | POST | `/upload` | BOM 文件上传 |
| `/bom` | POST | `/{id}/generate-issue` | 生成发料单 |
| `/issues` | POST | `/{id}/calculate` | 计算亮灯方案 (FIFO) |
| `/issues` | POST | `/{id}/assign` | 下发亮灯指令 |
| `/issues` | POST | `/{id}/confirm-pick` | 确认出库 (扫码) |
| `/inventory` | GET | `/?customer_id=&material_id=` | 库存查询 |
| `/xr` | POST | `/upload` | XR 点数上传 |
| `/xr` | POST | `/{id}/match` | 手动配对 |
| `/reports` | GET | `/daily?date=&customer_id=` | 日清表 |
